# Agent Reminders

These are project-local lessons and reminders. Preserve and extend this file.

- Domain before UI: domain -> workflow -> state -> architecture -> components -> visual UI.
- Prefer duplication over premature abstraction.
- Every state should have one owner.
- Refactor vertically and preserve behavior.
- Repository evidence outranks chat memory.
- Do not continue into the next Slice automatically.
- Update SDP first when scope or architecture changes.
